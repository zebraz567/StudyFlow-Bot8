## WHAT IS IT?
- It is an app that help students to organize their study plan!!!

## THE FEATURES:
- Add assignments
- List assignments
- Delete assignments
- Edit assignments
- Export assignments to JSON 
- And many more minor features

## TECH USED:
- Slack Bolt API
- JavaScript
- JSON File Storage
- A cup of coffee

## INSTALLATION
```bash
git clone https://github.com/zebraz567/study-flow-bot.git

cd study-flow-bot

npm install

npm start
```

## ENVIRONMENT VARIABLES:

Create a `.env` file:

```env
SLACK_BOT_TOKEN=
SLACK_APP_TOKEN=
SLACK_SIGNING_SECRET=
```

## COMMANDS
Command | Description |
|----------|-------------|
| add Title \| Deadline \| Hours \| Priority | Add assignment |
| list | Show assignments |
| done ID | Mark assignment complete |
| delete ID | Delete assignment |
| edit ID \| Title \| Deadline \| Hours \| Priority | Edit assignment |
| today | Today's study plan |
| upcoming | Upcoming assignments |
| stats | Study statistics |
| find keyword | Search assignments |
| export | Export assignments |
| help | Show help |
