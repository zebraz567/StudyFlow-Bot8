About

StudyFlow Bot is a Slack bot created using JavaScript, Node.js and Slack Bolt API. It helps students manage assignments, track deadlines and organize study schedules directly inside Slack.

✨ Features

  • ➕ Add assignments
  • 📋 List assignments
  • ✏️ Edit assignments
  • 🗑 Delete assignments
  • ✅ Mark assignments completed
  • 📅 Today's study planner
  • 📆 Upcoming deadlines
  • 📊 Statistics
  • 🔍 Search assignments
  • 📁 Export assignments

🛠 Tech Stack

  • JavaScript
  • Node.js
  • Slack Bolt API
  • JSON Storage
  • Git & GitHub
